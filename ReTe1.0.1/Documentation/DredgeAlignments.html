RetroTector

 

DredgeAlignments

This Executor in plugins is part of the package to make new Databases. It checks the consistency of all the alignment files in a Database subdirectory and constructs putative MotifGroups.txt (MotifGroupsCandidate.txt) and Genes.txt (GenesCandidate.txt) files from them, and a MakeSubGenesScript.txt file from which MakeSubGenes can generate a putative SubGenes.txt (SubGenesCandidate.txt). It also generates Dredge.txt with detailed findings.

The Database subdirectory where this is done must first be selected
as working directory.

DredgeAlignments should be run from a script looking something like this (second column in SubGenes is hotspot MotifGroup):

Executor: DredgeAlignments
SmallLimitFactor: 0.75
BigLimitFactor: 1.25
SubGenes::
5LTR 5LT
PBS  PBS
MA   MA1
CA   CA1
NC   NC1
DU   DU1
Prot PR2
RT   RT3
DL   DL1
IN   IN4
SU   SU3
TM   TM2
PPT  PPT
3LTR 3LT
::
Genes::
Gag  MA CA NC
Pro  DU Prot
Pol  RT DL IN
Env  SU TM
::


The range of distances between any two MotifGroups in all the alignment files (disregarding Suspect parts) is found. The distance ranges employed in the generated files are in principle calculated by multiplying the upper limit of this range by a factor and the lower limit by another.

These factors are the two parameters:

  • SmallLimitFactor: Lower limit factor. Default: 0.75.
  • BigLimitFactor: Upper limit factor. Default: 1.25.
