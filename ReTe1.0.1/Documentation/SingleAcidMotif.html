SingleAcidMotif.


This is a Motif subclass for Motifs hitting at single amino acids.
Its main use is as component of CoincidenceMotifs.

The MOTIFSTRING field of the defining line in Motifs.txt should contain a '!' defining the hotspot and acid symbols before or/and after it, specifying the valid acid at each distance from the hotspot. Only exactly matching acids are accepted.

The identifying string is '1A'.
