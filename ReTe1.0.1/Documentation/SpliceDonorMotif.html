SpliceDonorMotif


This is an OrdinaryMotif. Instances of this class score at possible splice donors. It uses the consensus:

  C/A A G!G T A/G A G T

where the hotspot is at the !.

Typically up to 2 deviations from this consensus are accepted. In that case, the raw score will equal the maximal raw score if there is no deviation, 2/3 of it if there is one and 1/3 if there are two.

The above limit may be changed to another value than 2, by setting the parameter 'MaxErrors' in Configuration.txt (using a SpliceDonorMotifdefaults parameter).

The identifying string is 'SPD'.
