The Database directory

The Retrotector/Database directory contains:

  • the file Configuration.txt
  • error log files
  • the file Properties.txt which you should usually leave alone
  • the subdirectory Ordinary
  • possibly more subdirectories with the same format as Ordinary, to be used for particular purposes.
Subdirectories contain a number of text files. The following are more or less essential to the function of RetroTector:
  • Motifs.txt
  • MotifGroups.txt
  • SubGenes.txt
  • Genes.txt
  • Inrvariants.txt
  • LTRIDData.txt
  • ORFHexamers.txt
  • NonORFHexamers.txt
  • RU5120NN.txt
  • Splitoct.txt
  • Transsites.txt
  • U3NN.txt
  • U3R120NN.txt
  • U5NN.txt
  • Alignment files (4 times the number of virus genera of them)
Others may needed if you use particular features.

Other subdirectories than Ordinary need not contain all files; missing files will be fetched from Ordinary instead.

If you know what you are doing, you may make minor changes in the files using any elementary text editor. You may also, at considerable inconvenience, generate your own Database subdirectories.


Inrvariants.txt

LTRIDData.txt

See LTRID.

ORFHexamers.txt

Contains a multiparameter with the key 'ORFHexamers'. It contains a list of DNA base hexamers known to occur frequently in retroviral ORFs. To the right of each is an indication of its ORF specificity, which is not used.

NonORFHexamers.txt

Contains a multiparameter with the key 'NonORFHexamers'. It contains a list of DNA base hexamers known to occur rarely in retroviral ORFs. To the right of each is an indication of its ORF specificity, which is not used.
