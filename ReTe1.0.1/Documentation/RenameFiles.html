RetroTector

 

RenameFiles


This Executor is a utility to rename or delete (not both functions simultaneously!) all files in a directory and its subdirectories fulfilling some criteria.

The directory to operate in must first be made the working directory of RetroTector.

There are four parameters:

  • Change: The string to change in file names.
    Default: 'Script.txt'.
  • Into: The string to change the above into.
    Default: 'Script_.txt'.
  • ChangeIfPresent: File name is changed only if it includes this string (unless it is empty, in which case all files are eligible).
    Default: '????'.
  • DeleteIfPresent: File is deleted only if its name includes this string. If it is empty, there is no deletion of files. A security question is asked before deleting is started (but not for each individual file).
    Default: ''.
To finish, the numbers of deleted and changed files are presented in the text window.
