Puteinview

This Executor shows a Putein, XXon or Xon file for inspection. It is normally run by selecting the file with the 'Choose script' button, and executing it.

A Putein file is an executor script created by ORFID (XXon and Xon files by XonID), with Puteinview as Executor. It should be noted that though ORFID forms an opinion about where the putein starts and ends, it usually constructs a longer putein than that, in case its opinion is wrong.

A Putein, XXon or Xon file has two or three parameters actually used by Puteinview:

  • 'FontSize'. A singleparameter specifying the size of the displayed characters. Default = 14.
  • Hits: (in Puteins) A multiparameter with descriptions, one per line, of the MotifHits ORFID used to make this Putein.
  • 'Putein', or 'Xon' in a XXon or Xon file. A multiparameter with five (very long) lines. The second of these shows the Putein (Xon) sequence with standard amino acid symbols. 'z' means stop codon, 'x' ambiguous acid (because of ambiguous base in the DNA). Uppercase characters (in a Putein) mark acids which were exactly matched in the Alignment. The third line shows the reading frame used at each acid. The first line (in a Putein) indicates where the Motif hits in the 'Hits' parameter (numbered from 1 upwards) are relative to the second line. In the fourth line '>' and '<' are used to indicate ORFID's opinion about start and end of the putein (the arrows are at the acid before and after the estimated first and last acid, respectively). '!' indicate hits by the Kozak consensus motif. The fifth line shows base position in the DNA at intervals of approximately 90.
Puteinview shows a window with two frames, the upper one showing the contents of the file, the lower he contents of 'Putein' or 'Xon', using a fixed-width font.

If you should want to run Puteinview interactively, the parameter window has two parameters, 'FontSize' (see above) and 'PuteinFile', with obvious meaning.

There are also a number of parameters not used by Puteinview, but for documentation. They are (most of them are not present in XXon or Xon files):

  • NumberOfHits: The number of MotifHits ORFID could localize in the Alignment.
  • LeadingInfo: A somewhat cryptic description of the leading putein segment.
  • LeadingInfo2, LeadingInfo3 etc: Similar descriptions of all the possible leading putein segments considered by ORFID. Only if ORFID was run with the Debugging option.
  • TrailingInfo: A somewhat cryptic description of the trailing putein segment.
  • TrailingInfo2, TrailingInfo3 etc: Similar descriptions of all the possible trailing putein segments considered by ORFID. Only if ORFID was run with the Debugging option.
  • Trouble: Copied from the Trouble parameter in the ORFID script. If present, it describes some problem (eg BrokenChain) encountered by RetroVID, suggesting that the output of ORFID may not be meaningful.
  • Gene: Gag, Pro, Pol or Env
  • Genus: A character describing the virus genus.
  • DNAFile: The relevant DNA file.
  • EstimatedStartPosition: The base position of the start of the putein, in the opinion of ORFID.
  • EstimatedLastPosition: The base position of the end of the putein (third base of last codon), in the opinion of ORFID.
  • LengthInside: (Length in XXon or Xon files) Number of acids in the putein between the suggested start and end.
  • LengthTotal: Overall number of acids in the putein.
  • AlignedAcids: Number of acids determined in accordance with the Alignment.
  • AverageScoreInside: A number between 0 and 1, indicating the quality of the putein between the suggested start and end.
  • AverageScoreTotal: A number between 0 and 1, indicating the quality of the overall putein.
  • MostUsedRow: The number and description of the row in the Alignment ORFID came to rely most on.
  • StopCodonsInside (Stops in XXon or Xon files): The number of stop codons in the putein between the suggested start and end. In parentheses, the number of stop codons in each reading frame over the same region.
  • StopCodonsTotal: The number of stop codons in the overall putein.
  • AmbiguousAcidsInside: Number of ambiguous acids between the suggested start and end.
  • AmbiguousAcidsTotal: Number of ambiguous acids in the overall putein.
  • ShiftsInside(Shifts in XXon or Xon files): Number of frame shifts in the putein between the suggested start and end.
  • ShiftsTotal: Number of frame shifts in the overall putein.
  • LongestRun: The length of the longest run of acids in the same reading frame between the suggested start and end, and its start position.
  • LongestORF: The starting position and content of the codon sequence common between the putein and the longest ORF having at least one codon in common with the putein. If the ORF extends beyond this sqeuence at either end, it is marked with '<<' or '>>' respectively.
  • CodonSequence: The bases actually corresponding to the acids between the suggested start and end,
