EnvProcessor


This plugin ChainProcessor creates a script for EnvTracer if a Chain
  • Has a 3'LTR
  • Has a hit in the IN SubGene
There is one parameter, adjustable through Configuration.txt:
  • ProcessorPriority: Priority relative to other ChainProcessors.
    Default: -1.
