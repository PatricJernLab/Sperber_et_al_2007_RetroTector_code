PseuGID


PseuGID is an Executor to identify a Chain as a possible processed pseudogene. It searches for a structure of the type
TSD(target site duplication) - R....R - A-rich tail - TSD. It is under development and at the moment not satisfactory.

PseuGID is normally run using a script generated by RetroVID. It contains information about DNA, Database and the parent Chain.

If it finds evidence of the pseudogene structure, PseuGID outputs a file with name Pseudogene...txt, a parameter file. It has a multiparameter describing the Chain (see Chainview) and a number of singleparameters, most of them with self-explaining names. A few may require explanation:
  • R1hotspot: The position of the third base (aataaa) in the polyadenylation signal in the first R.
  • R2hotspot: Similarly for the second R.
  • Score: A goodness estimate, calculated in a way too complex to explain.
The ends of the R repeats are primarily defined on criteria of similarity. In the R1 and R2 strings, blanks mark the ends calculated by RMotif. Since RMotif is at present not satisfactory, they should not be taken too seriously.


The parameter of PseuGID is:
  • Tolerance: Tolerance parameter in R aligning, similar to LTRepTolerance in LTRID. Default: 3.
