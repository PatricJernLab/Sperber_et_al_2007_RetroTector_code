RetroTector

 

Plugins


The 'plugins' directory should be present alongside RetroTector.jar, but does not need to contain anything. It offers a convenient way to add extra facilities in the form of Java .class files. At present, these may be of four kinds:
  • Executors
  • Motifs
  • ChainProcessors
  • Brooms
At startup, RetroTector searches 'plugins' for classes of these kinds, listing them internally for future reference.

If a class is present both in RetroTector.jar (in the package 'builtins') and in 'plugins', the latter will be used.

'plugins' may also contain other .class files defining internal classes belonging to one of the above, but this should not be overused, since they are apt not to be removed if the parent class is removed.

At present, the standard plugins contains the following files which are either of little interest to most users, or on trial (the first two are recommended):

    EnvProcessor
    EnvTracer
    CheckDatabase
    DredgeAlignments
    LTRAlign
    MakeSubGenes
    PBSFind
    RawScoreStatistics
    RoughcutDNA
