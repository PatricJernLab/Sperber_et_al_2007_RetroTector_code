RetroTector

 

Virus genus


At present (but it may be easily changed, should the need arise), RetroTector recognizes 9 virus genera, each with a single-letter symbol:
  • A: alpharetroviruses
  • B: betaretroviruses
  • C: gammaretroviruses
  • D: deltaretroviruses
  • E: epsilonretroviruses
  • L: lentiviruses
  • S: spumaviruses
  • G: Gypsy elements
  • O: Copia elements
Into RetroTector is built a matrix of rough measures of relatedness between the genera, which looks like this:

   A B C D E L S G O
 A  3 2 1 1 1 0 0 0 0
B  2 3 1 1 1 0 0 0 0
C  1 1 3 2 1 0 1 0 0
D  1 1 2 3 1 1 0 0 0
E  1 1 1 1 3 0 0 0 0
L  0 0 0 1 0 3 0 0 0
S  0 0 1 0 0 0 3 0 0
G  0 0 0 0 0 0 0 3 0
O  0 0 0 0 0 0 0 0 3

RVectors

In order to keep track of the virus genus  RetroTector uses RVectors. An RVector is a vector with as many components as there are virus genera. Each virus genus is associated with an RVector of unit length. These are adjusted so that the scalar products between pairs of them correspond to the matrix above.

Motif hits are associated with a vector: the RVector of the virus genus of the Motif times the hit score. SubGene hits and Chains are also associated with vectors which are the sum of the vectors of its elements, multiplied by some correction factors.
