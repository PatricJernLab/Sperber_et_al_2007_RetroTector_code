RetroTector

 

The file MotifGroups.txt


This parameter file in Database subdirectories specifies MotifGroups.

A MotifGroup is a collection of Motifs expected to match at the same position in a retrovirus. It may include Motifs from different virus genera. It belongs to a SubGene .

MotifGroups.txt contains a number of singleparameters, one for each MotifGroup. The key of the parameter is the name of the MotifGroup.

The contents of the parameter must contain at least two blank-separated fields:

  • The name of the SubGene to which the MotifGroup belongs.
  • A range specifier. It specifies the acceptable range for the distance between the hotspot of a Motif hit by a Motif belonging to this MotifGroup, and the hotspot of the SubGene hit in which it is included.
There may also be an arbitrary number of subsequent blank-separated fields consisting of a MotifGroup name and a range specifier separated by a ':' (no blanks allowed) which specify the acceptable range between the hotspot of a Motif hit by a Motif belonging to this MotifGroup, and the hotspot of a Motif hit by a Motif belonging to another MotifGroup in the same SubGene hit.
