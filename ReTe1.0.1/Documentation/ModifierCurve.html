ModifierCurve

This Executor shows a curve over a Modifier along a DNA strand. It is normally started interactively, and has two parameters:

  • 'DNAFile'. The DNA file to use.
    Default: Directory name + .txt.
  • 'Strand'. 'Primary' or 'Secondary'.
    Default: Primary.
At the bottom of the parameter window, there is a pop-up menu, where a Modifier may be selected.

The curve is shown in a separate window, with the maximal value achieved by the Modifier in the upper left corner. If the Modifier is of such a nature that information about reading frame is meaningful, separate curves for the reading frames will be shown in different colours.

A button at the bottom of the graphics window allows saving the graphics as a (very oblong) PNG, EPS or JPEG image file.
