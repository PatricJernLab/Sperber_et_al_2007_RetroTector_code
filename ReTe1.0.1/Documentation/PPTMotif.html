RetroTector

 

PPTMotif


This is an OrdinaryMotif, evaluating similarity to polypurine tracts.

Its identifying string is 'PPT'.

There are three named parameters, whose values may be redefined in Configuration.txt :

  • LowLimit (normally = 20)
  • HighLimit (90)
  • TgBonus (5)
The raw score is calculated thus: if
  • the targeted base is not 'a' or
  • the preceding base is 'a' or
  • the next base is not 'a' or 'g'
no hit is scored. Otherwise an end position is located as the position before the second non-ag base downstream. The raw score is calculated as the sum of contributions from each 'a' between the start and the end positions, the contribution from each 'a' being the number of 'g' after it, up to the end position. If a 'tg' pair is then found within 5 positions downstream from the end position, TgBonus is added to the raw score.

If the raw score is > LowLimit, a hit is registered, with final score proportional to (raw score - LowLimit), such that a raw score of HighLimit (or possibly more) yields a final score equal to the value in the MAXSCORE field in Motifs.txt.
