RetroTector

 

LTRview


This Executor shows a graphic view of the contents of an LTR candidate, found by LTRID and defined in a RetroVID script. It is in a state of development.

There is one parameter:

  • InputFile: The name of the file with LTR candidate descriptions. Default: RetroVID_001Script.txt.
When run, LTRview shows a menu with the LTR candidates available, and one of them, usually the first, is displayed in a window.

The window consists of a number of narrow horizontal strips showing various kinds of information, each with a name to the left. There are also buttons to save the display as a (rather unwieldy) PNG, EPS or JPEG file.

The horizontal axis is defined by the 'Sequence' strip, which shows all the bases in the LTR, plus ten leading and ten trailing. Position numbers are shown at intervals. Vertical black lines (going through the entire diagram) mark the beginning, hotspot (T in AATAAA or equivalent) and end of the LTR. Vertical blue lines mark the suggested start and end of the R part of the LTR.

The other strips are at present (see LTRID for some clarification), with the ones related to internal analysis of the LTR above and those related to LTR identification below the 'Sequence' strip:
  • MEME50: The score of the MEME50 weight matrix (between min and max values). Maxima are marked with a vertical red line.
  • TATAA: The score of the TATAAA box weight matrix (between min and max values). Maxima are marked with a vertical red line.
  • RU5 neural net: The score (between 0 and 1) of the neural network defined by the file RU5120NN.txt. Maxima are marked with a vertical red line.
  • Inrsites: Occurrences of any of the motifs defined in Inrvariants.txt in Database.
  • U3R neural net: The score (between 0 and 1) of the neural network defined by the file U3R120NN.txt. Maxima are marked with a vertical red line. Values are associated with the position at the end of the net.
  • REndModifier: The value of REndModifier.
  • U5 neural net: The score (between 0 and 1) of the neural network defined by the file U5NN.txt. Maxima are marked with a vertical red line.
  • U3 neural net: The score (between 0 and 1) of the neural network defined by the file U3NN.txt. Maxima are marked with a vertical red line.
  • Transcr. sites: Shows matches with the transcription sites in the file Transsites.txt.
  • Direct repeats: Shows up to 20 direct repeats.
  • TransSiteModifier: The value (between min and max) of the TransSiteModifier.
  • CpGModifier: The value (between min and max) of the CpGModifier.
  • SplitOctamerModifier: The value (between min and max) of the SplitOctamerModifier.
